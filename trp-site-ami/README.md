# TRP - Transport

Bienvenue sur le dépôt du projet **TRP**. Il s'agit d'un site web vitrine conçu pour présenter un service de transport. Le site est construit de manière statique pour assurer fluidité, légèreté et simplicité de déploiement.

## 🚀 Fonctionnalités et Structure du Site

Le projet est divisé en plusieurs pages pour offrir une navigation claire et intuitive aux visiteurs :

- **`index.html`** : Page d'accueil du site.
- **`presentation.html`** : Présentation détaillée de l'entreprise et des services proposés.
- **`tarif.html`** : Grille tarifaire des différentes prestations de transport.
- **`planning.html`** : Horaires, plannings et disponibilités.
- **`galerie.html`** : Galerie d'images illustrant les services ou les véhicules.
- **`contact.html`** : Informations de contact pour joindre l'entreprise.

## 🛠️ Technologies Utilisées

Ce projet repose sur des technologies web standards :
- **HTML5** : Pour la structure et la sémantique des différentes pages.
- **CSS3** : Pour la mise en page, le design et l'adaptation visuelle (via le fichier `styles.css`).
