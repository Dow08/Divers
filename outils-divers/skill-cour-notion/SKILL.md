# Cyber Course Builder - Complete Workflow

## What This Skill Does

This skill transforms ANY cybersecurity content into a professionally structured course and **automatically saves it to your Notion database**.

**Three-step process :**
1. **Accept content** in any format (text, PDF, Word, URL, fragments)
2. **Structure it** according to the 9-section cybersecurity course format
3. **Save directly to Notion** with metadata, tags, and full verification

## Input Formats Supported

- ✅ Plain text (copy-paste)
- ✅ PDF files
- ✅ Word documents (.docx)
- ✅ URLs (skill will fetch content)
- ✅ Fragmented notes or snippets
- ✅ Multiple sources combined

## The 9-Section Structure

Every course follows this exact format:

1. **Titre** - Clear, precise course title
2. **Contexte** - Security context (pentest, blue team, admin, forensic, etc.)
3. **Définition** - Simple, clear explanation
4. **Points clés** - Essential concepts (bolded)
5. **Fonctionnement** - Structured explanation/steps
6. **Commandes / Techniques** - Practical commands with code blocks
7. **Cas d'usage** - Real-world examples
8. **Risques / Sécurité** - Vulnerabilities and threats
9. **Résumé** - 3-5 line quick reference

## Complete Workflow: From Content to Notion

### Step 1: Provide Content
You give the skill raw cybersecurity content in any format.

**Examples:**
- "I have notes on SQL injection, here's the content..."
- "Upload this PDF about wireless hacking"
- "Structure this article: https://example.com/article"
- Fragmented bullet points or mixed formats

### Step 2: Skill Transforms
The skill:
1. Parses all input formats (PDF extraction, URL fetching, plain text)
2. Combines multiple sources if provided
3. Applies the 9-section structure
4. Enriches with examples and technical details
5. Formats for Notion (markdown, code blocks, metadata)

### Step 3: Automatic Notion Integration
The skill **directly creates a page in Notion** with:
- ✅ All 9 sections properly formatted
- ✅ Domain/topic tags (pentest, blue team, forensics, etc.)
- ✅ Metadata for searchability
- ✅ Syntax-highlighted code blocks
- ✅ Proper hierarchy and formatting
- ✅ Ready-to-use in your "cours cybersécurité" database

### Step 4: Quality Verification
Before finishing, the skill:
1. **Verifies completeness** - All 9 sections present and substantive
2. **Verifies structure** - Proper headings, formatting, code blocks
3. **Verifies searchability** - Metadata and tags are correct
4. **Verifies organization** - Course is in correct Notion location
5. **Reports results** - Shows you what was saved and any issues found

## Notion Integration Details

### Database Location
- **Database:** cours cybersécurité
- **Properties Created:**
  - Title (course name)
  - Domain (pentest, blue team, forensics, threat analysis, admin, etc.)
  - Difficulty (introductory, intermediate, advanced)
  - Category (Web Security, Network, Infrastructure, etc.)
  - Tags (searchable keywords)
  - Created Date (automatic)
  - Last Updated (automatic)
  - Language (French/English)

### Notion Formatting
- Page title automatically set

- Code blocks properly formatted with syntax highlighting
