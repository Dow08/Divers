# SORTIA — Où créer chaque clé API

## Supabase (backend + base de données)
1. https://supabase.com → New Project → Région EU Frankfurt
2. Settings → API → Copier `Project URL` et `anon key`

## Google Gemini (IA)
1. https://aistudio.google.com/apikey → Create API key
2. La même clé fonctionne pour Flash et Pro

## Yousign (signature électronique)
1. https://yousign.com → Créer un compte
2. Settings → API Keys → Create (mode Sandbox pour les tests)
3. URL Sandbox : `https://api-sandbox.yousign.app/v3`

## Stripe (paiements)
1. https://dashboard.stripe.com/register
2. Developers → API Keys (mode test par défaut)
3. Créer les produits : Starter (7,90€), Pro (14,90€), Expert (24,90€), Business (44,90€)

## Sentry (monitoring erreurs)
1. https://sentry.io/signup/ → Gratuit (5K events/mois)
2. New Project → Flutter → Copier le DSN

## Snyk (scan vulnérabilités)
1. https://app.snyk.io/login → Signup with GitHub
2. Settings → General → Auth Token
